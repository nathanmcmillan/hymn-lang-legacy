Hymn is a programming language designed to make writing simple imperitive programs easy. It compiles to efficient, readable C code.

Download Hymn
type vec<t>
    x t
    y t
    z t

hello(flag bool) maybe<vec<float32>>
    if flag
        return vec(1.2, 2.3, 3.4)
    else
        return none

main
    v = hello(true)
    match v
        some => echo(v.x)
        none => _
